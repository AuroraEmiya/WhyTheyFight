# OPEN_QUESTIONS.md

本文件用于记录 WhyTheyFight 当前未决设计问题。

目的：

```text
1. 防止开发者或 agent 在设计未明确时自行补规则；
2. 让玩法决策集中记录；
3. 为后续迭代提供讨论清单。
```

---

# 1. 回合结构

```text
Q1. 是玩家 A 完整行动后玩家 B 行动，还是所有玩家同时下达指令后统一结算？
Q2. 每回合是否有阶段？
    - 资源阶段
    - 部署阶段
    - 行动阶段
    - 战斗结算阶段
    - 结束阶段
Q3. 每个单位每回合只能执行一个主要行动，还是可以通过行动点多次操作？
Q4. 行动点每回合如何恢复？
Q5. 行动点是否可以储存到下一回合？
Q6. 回合结束条件是什么？
```
